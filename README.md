# Guest Book

## Project Documentation

### Overview
The Guest Book application allows users to sign in and leave comments or feedback. It provides a simple interface for visitors to share their thoughts and for the administrators to manage entries.

### Features
- Easy user sign-in and sign-out.
- Simple interface for submitting feedback.
- Admin panel for managing entries.

### Technologies Used
- HTML, CSS, JavaScript for front-end development.
- Node.js and Express for back-end API.
- MongoDB for data storage.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ProRow15/guestbbook.git
   ```
2. Navigate to the project directory:
   ```bash
   cd guestbbook
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```

### Usage
- Open your web browser and go to `http://localhost:3000` to access the application.
